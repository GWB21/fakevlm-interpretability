7개 카테고리 모두 정상 생성되었습니다. 이미지에서 즉시 관찰되는 현상을 설명합니다.

## 시각화에서 관찰된 주요 패턴

### DOC 카테고리
- Saliency가 **이미지 내 삽입된 사진/그래픽 영역**에 집중되어 있습니다. 순수 텍스트 블록보다 삽입 이미지 근방에 attention이 높습니다.
- 거의 모든 행이 `★` (Language Bias) — 마스킹 전후 예측이 일치합니다. 이는 HPF+DC에서 99% 정확도를 유지한 것과 일관된 결과로, 모델이 텍스트 문서임을 "보지 않고도" 언어 사전으로 판단하는 경향이 있습니다.

### DEEPFAKE 카테고리
- Saliency가 **얼굴 중앙(눈·코 주변)**에 일관되게 집중됩니다. FAKE와 REAL 모두 동일한 패턴이나, FAKE에서 더 집중적입니다.
- `REAL→FAKE` 전환 사례가 있어 마스킹이 예측을 바꾸는 케이스가 존재합니다.

### SATELLITE 카테고리
- Saliency가 **이미지 전체에 분산**되어 특정 영역에 집중되지 않습니다. FFT 실험에서 satellite가 필터링 후 랜덤 수준으로 떨어진 결과와 일치합니다. 모델이 위성 이미지에서 어디를 봐야 하는지 명확한 기준이 없음을 시사합니다.

## "Hallucination인가?" 에 대한 정확한 답

**절반만 맞습니다.** 더 정확하게는 두 가지 현상이 혼재해 있습니다.

### ① 95.68%가 "예측 불변" — 이것이 Language Bias

```
[불변+정답]  94.5% → 모델이 맞췄는데 마스킹해도 여전히 맞춤
[불변+오답]   1.2% → 모델이 틀렸는데 마스킹해도 여전히 틀림 ← 순수 언어 편향
```

"FakeVLM이 가장 많이 보는 영역 20%를 가렸는데도 예측이 안 바뀐다"는 것은 모델이 특정 시각 증거를 실제로 필요로 하지 않는다는 의미입니다. 즉 시각 정보 없이도 **언어 사전(Language Prior)** 만으로 판단하는 비율이 매우 높다는 증거입니다.

그러나 이건 **전통적 Hallucination**(없는 것을 본다)과는 다릅니다. 오히려 **"보지 않고도 맞추는"** 현상입니다.

### ② 그러나 4.1%는 마스킹 후 정답→오답으로 전환

```
[변화→오답]  4.1% (206건) → 시각 패치 제거가 실제로 성능을 떨어뜨림
```

이 206건은 모델이 해당 패치를 **진짜로 시각적 근거로 사용**하고 있었다는 증거입니다. 완전한 언어 편향은 아닙니다.

## 최종 해석 (논문 서술 관점)

> FakeVLM은 대부분의 카테고리(satellite, doc, animal 등)에서 **시각적으로 가장 중요한 패치를 제거해도 예측을 유지**하는 강한 Language Prior 의존성을 보입니다. 이는 Hallucination이라기보다 **텍스트 질문 패턴으로부터 정답을 추론하는 Language Bias**에 가깝습니다. 다만 deepfake 카테고리에서는 마스킹 후 12.4%p 하락이 관찰되어, 얼굴 위변조 탐지에는 실제 시각적 근거가 존재함을 확인했습니다.

매우 핵심적인 방법론적 질문입니다. 찬성하는 이유와 주의해야 할 지점을 함께 분석합니다.

## Cross-Dataset 검증이 필요한 이유 (찬성 논거)

### 1. 현재 실험의 근본적 한계: In-Distribution 테스트

FakeVLM은 FakeClue로 학습되었고, 현재 실험도 FakeClue test set에서 수행되었습니다.

```
훈련: FakeClue train → 모델이 FakeClue의 언어 패턴, 이미지 통계, 질문 형식을 학습
검증: FakeClue test  → 동일 분포에서 테스트
```

이 상황에서 두 실험에서 관찰한 현상들이 **진짜 기작인지, 훈련 데이터 암기인지** 구분이 불가능합니다.

### 2. 실험별 구체적 문제

**실험 1 (FFT) 관점:**
- LPF 조건에서 정확도가 60%로 급락했는데, 이것이 "FakeVLM이 고주파에 의존한다"는 증거인지, 아니면 "FakeClue 이미지들이 LPF 후 FakeClue 훈련 분포를 벗어나서 예측을 못 하는 것"인지 구분이 불가능합니다.
- Cross-dataset에서도 동일한 LPF 패턴이 나온다면, 주파수 의존성이 일반적 기작임을 확인할 수 있습니다.

**실험 2 (Hook) 관점:**
- Language Bias Rate 95.68%가 진짜 "언어 사전 의존"인지, FakeClue에 특화된 텍스트 패턴 암기인지 불명확합니다.
- 모델이 FakeClue의 질문 형식 ("Does the image looks real or fake?")에 과적합되어 시각 정보 없이도 맞출 수 있는 편향이 학습되었을 가능성이 있습니다.

## 그러나 다른 연구 질문에 답하는 것임을 인식해야 함

| 질문 | 적합한 실험 |
|---|---|
| "FakeVLM은 내부에서 무엇을 보고 판단하는가?" | In-distribution (현재 실험) |
| "FakeVLM의 기작이 새로운 데이터에도 유효한가?" | **Cross-dataset 필요** |

현재 실험은 첫 번째 질문에 대한 답입니다. Cross-dataset은 두 번째 질문을 추가로 답하는 것이므로 **결과가 다르더라도 서로 모순이 아닙니다.**

## 결론적 판단

Cross-dataset 검증은 **Top-tier 논문 요건**이자 현재 분석의 약점을 보완하는 매우 타당한 제안입니다.

특히 두 가지 발견이 Cross-dataset에서도 유지된다면 논문의 기여도가 크게 올라갑니다:

1. **Language Bias 95.68%** — 본 모델의 구조적 특성인가, FakeClue 암기인가
2. **doc 카테고리 HPF+DC 99.13%** — 고주파 텍스트 엣지가 보편적 진단 특징인가

반면 Cross-dataset에서 결과가 달라진다면 그것 자체가 "FakeVLM은 FakeClue에 특화된 편향을 학습했다"는 새로운 발견이 되므로, **어느 방향이든 의미 있는 결과가 나옵니다.**
